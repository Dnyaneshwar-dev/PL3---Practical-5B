Problem Statement

Write a code in any programming language that converts Centigrade to Fahrenheit

Code

        
          #include <bits/stdc++.h>
          using namespace std;
          int main()
          {
              int centigrade;

              cout << "Enter Temperature in centigrade : ";
              cin >> centigrade;

              int fahrenheit = (int)((double)9/5 * centigrade) + 32;

              cout << fahrenheit;

          }

Sample Input

        
          45

Sample Output

        
          113

Time Complexity

        
          O(1)
